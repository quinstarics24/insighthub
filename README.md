# InsightHub

InsightHub is a web-based platform for managing posts, resources, and messages. It includes an admin panel for managing content, viewing messages, and tracking resources.

## Features

- Admin dashboard with total posts, resources, and messages
- CRUD operations for posts and resources
- Contact form for visitors
- Messages management in admin panel
- Responsive sidebar and footer layout
- File uploads for posts and resources

## Technologies Used

- PHP
- MySQL (via XAMPP)
- Bootstrap 5
- HTML & CSS
